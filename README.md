# 💕 照片墙爱心排版

一个使用数学心形公式生成完美爱心照片墙的 Web 应用。

## ✨ 功能特点

- 📸 **拖拽上传** - 支持从本地拖拽图片或点击选择（支持多选）
- 💕 **爱心排版** - 一键生成爱心形状照片墙
- 🔄 **循环复用** - 照片数量不足时自动循环复用
- 📱 **移动端适配** - 完美支持手机浏览器和微信
- 📥 **高清下载** - 生成 1000x1000 高清 PNG 图片
- 🎨 **简约设计** - 清新温馨的配色方案

## 🚀 使用方式

### 本地运行

直接用浏览器打开 `index.html` 即可运行，无需任何服务器或依赖。

```bash
# 方式 1: 直接用浏览器打开
open index.html

# 方式 2: 使用简单 HTTP 服务器
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```

### 在线访问

访问部署在 Vercel 的在线版本使用。

## 📱 移动端使用

1. 在手机浏览器或微信中打开链接
2. 点击上传区域从相册选择照片
3. 点击"一键排版"生成爱心
4. 点击下载保存高清 PNG

## 🛠️ 技术栈

- 纯 HTML/CSS/JavaScript
- 参数方程生成心形曲线
- Canvas API 绘制下载图片
- 响应式设计适配移动端

## 📝 使用说明

1. **上传照片** - 拖拽或点击选择多张照片
2. **一键排版** - 点击按钮生成爱心形状
3. **下载保存** - 生成透明背景高清 PNG

## 💡 提示

- 支持任意数量的照片（1 张也能生成完整爱心）
- 照片会自动从中心裁剪为正方形，保持圆形显示不变形
- 下载的图片为透明背景，适合各种场景使用
